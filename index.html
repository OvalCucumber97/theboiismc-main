Philosophy

The modern web has become centralized, tracked, and monetized. My goal is to reclaim digital sovereignty by self-hosting the essential tools I use every day.

This project focuses on Zero-Knowledge architecture, Open Source software, and No-BS usability. If you stumbled here by accident, feel free to look around, but know that this infrastructure is built for a specific purpose, not for mass consumption.
